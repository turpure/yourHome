关于我们

      上海幽然实业有限公司于2009年开始创业，因公司迅猛发展成为ebay超级卖家。有高效的销售团队和独立宽敞的仓库物流中心。公司专注于全球电子商务和在线零售业务，已经在eBay、速卖通、Wish、亚马逊、Joom全球电子商务平台拥有多个超级卖家网店 。经营数千种服饰、箱包、饰品、鞋帽、户外、时尚家居、3C数码电子用品，热销澳洲、北美、欧洲以及其他主要国家，日销量数千单！公司坐落于上海浦东金桥加工开发区内，周边资源丰富。 公司秉承“以人为本，愉快工作”的宗旨，致力营造轻松、活跃的工作氛围。不间断地组织员工出游或拓展活动，极大地推进了团队协调作用的强大和重要性！倡导高效、公平、付出与回报成正比的多元化绩效管理机制!而合理的利润分享机制则是我们最大的核心竞争力!
      如果你想挑战自己，就请加入我们!如果你想成就自己，也请加入我们！我们是一群年轻、充满朝气的团队！我们是一面永不磨灭的番号！

PHP工程师

  • 经验要求：2-3年经验；
  • 工作地点：浦东新区新金桥路1295号1号楼501室；

  • 负责将公司现有Excel模型及公式编写成程序；
  • 按照公司业务需求完成相关项目功能的开发和编程工作；

  4. 有2-3年左右php等语言开发经验，具有使用PHP语言开发数据库软件的工作经验优先；
  5. 有良好的学习、沟通能力，能承受一定的工作压力；

Amazon/Wish/eBay/Joom/速卖通销售专员

  • 经验要求：1-3年；
  • 工作地点：浦东新区新金桥路1295号1号楼501室；

  • 编辑产并优化商品信息，进行有效推广，取得良好业绩；
  • 负责网店产品销售工作，及时反馈商品销售信息，达成部门制定的账号销售目标；
  • 处理所管理账号的客户邮件，解决客户纠纷和中差评，并通过邮件积极与客户沟通，达成销售；
  • 上级安排的其他临时性工作。
任职资格
  1. 大专以上学历，国际贸易及商务英语等相关专业，大学英语四级以上；
  2. 对跨境电商有一定的了解，对该行业感兴趣，有较强的学习能力；
  3. 有B2B及B2C网站操作及推广经验； Amazon平台、eBay网店、速卖通、Wish经验者优先。

产品开发专员

  • 所属部门：产品部；
  • 经验要求：0-3年，应届生亦可；
  • 工作地点：浦东新区新金桥路1295号1号楼501室；

  1. 能充分认同公司的价值观和用人观；

  3. 有一定的数据分析能力，熟练使用Word、 Excel等办公软件进行数据调研；
  4. 工作细致认真，责任心强，具有较强的沟通能力、谈判能力和成本意识。

ebay/SMT/amazom/wish销售主管

  • 所属部门：销售部
  • 经验要求：3年以上，至少1年以上管理经验
  • 工作地点：上海
  • 负责任一平台所有账号，及时向上反馈商品销售信息，达成公司制定的各阶段销售目标；
  • 负责管理任一销售团队，对新人进行岗前培训，定期开展岗位技能培训；
  • 负责检查下属日常工作，并给出相应的有针对性的指导；
  • 负责分析各账号的销售情况，针对销售问题作出分析并给出解决方案；
  • 时时跟进平台最新政策，了解行业动态，对部门销售策略及时做出有效调整；
任职资格
  1. 大专以上学历，国际贸易或商务英语等相关专业，大学英语四级以上；
  2. 三年以上相关平台销售经验，至少一年以上的团队管理经验；
  3. 有良好的沟通能力和团队协调能力，善于进行团队激励；
  4. 熟悉相关平台政策和规定，能妥善解决可能出现的各种争议，最大程度的维护公司利益。

线下采购/驻厂跟单/服饰采购

  • 所属部门：采购部；
  • 经验要求：2-5年工作经验；
  • 工作地点：新金桥路1295号1号楼501室；
  • 深度挖掘公司产品的源头生产供应商，降低采购成本；
  • 熟悉生产环节，把控生产成本，定期与供应商进行降低成本的沟通和分析，并进行品质控制；

  3. 工作细致认真，责任心强，具有较强的沟通能力、谈判能力和成本意识。
  4. 男性优先，适应长期出差或驻厂
